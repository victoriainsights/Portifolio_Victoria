# 👨‍🚀 Desafio de Engenharia de Prompt: Astronauta Barroco em Marte

## 📝 Descrição do Projeto
Este projeto consiste na aplicação de técnicas de **Engenharia de Instrução** para gerar uma imagem específica utilizando a IA **Qwen**. O objetivo foi alcançar a representação de um "Astronauta em Marte com estilo artístico Barroco", respeitando o limite estrito de apenas 5 iterações (tentativas).

Desenvolvido como parte da disciplina de **Engenharia de Prompt e Aplicações em IA**, o foco foi o ciclo de análise crítica, integrando aprendizados de cada erro para ajustar o tom, a iluminação (chiaroscuro) e a estética da resposta gerada.

---

## 🚀 Tecnologias Utilizadas
* **Modelo de IA:** Qwen LLM.
* **Ferramenta:** Qwen Mobile.
* **Técnica:** Engenharia de Prompt com foco em Iteração e Refinamento.

---

## 📊 Resultados e Aprendizados
O projeto atingiu o objetivo visual esperado dentro do limite proposto:
* **Acurácia Visual:** O modelo conseguiu integrar elementos futuristas do traje espacial com a técnica de luz e sombra típica do Barroco.
* **Redução de Ruído:** Aprendizado focado na remoção de termos genéricos que confundiam a IA, substituindo-os por descritores de estilo artístico precisos.
* **Evolução de Prompts:** O uso de feedbacks corretivos permitiu que a IA ajustasse a textura da armadura e a ambientação marciana conforme as tentativas avançavam.
