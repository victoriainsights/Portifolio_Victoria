# 📊 Batalha de Modelos & Engenharia de Prompt (XML)

## 📝 Descrição do Projeto
Este projeto consiste em uma análise crítica para avaliar a precisão técnica e a conformidade de diferentes modelos de linguagem (**LLMs**) em relação a um conjunto estrito de instruções estruturadas[cite: 1]. O foco principal foi o desenvolvimento da metacognição ao identificar os pontos fortes e as limitações de cada arquitetura de IA para o desenvolvimento front-end[cite: 1].

---

## 🎯 Objetivos da Atividade
* **Avaliação Técnica:** Medir a fidelidade das IAs ao processar tags XML específicas[cite: 1].
* **Análise de Performance:** Comparar o consumo de tokens e a verbosidade de cada modelo[cite: 1].
* **Rigor de Design:** Verificar o cumprimento de diretrizes de layout responsivo, tipografia e paleta de cores (#F9FAFB, #3B82F6, #8B5CF6).

---

## 🚀 Protocolo de Execução
* **Prompt Estruturado:** Utilização de tags XML como `<tarefa>`, `<diretrizes_design>` e `<obrigatoriedades_tecnicas>` para guiar o modelo.
* **Modelos Testados:** ChatGPT, Gemini, Claude, Qwen, DeepSeek, Grok e Maritaca[cite: 1].
* **Tarefa Prática:** Geração de uma página HTML Single Page para o tema "Busca Inteligente de Estágios".

---

## 📊 Quadro de Análise Comparativa
De acordo com os dados coletados durante o experimento:

| Modelo | Precisão do Prompt | Consumo de Tokens | Observação Técnica |
| :--- | :--- | :--- | :--- |
| **DeepSeek** | Excelente | 3.200 | Entregou exatamente o solicitado. |
| **ChatGPT** | Insuficiente | 900 a 1.100 | Resultado muito simples. |
| **Gemini** | Boa | 1.350 | Entregou o que foi solicitado. |
| **Claude** | Regular | 4.800 | Verbosidade altíssima; fugiu do pedido. |
| **Qwen** | Regular | 1.300 | Estrutura básica demais. |

---

## 🧠 Reflexão Crítica
* **Compreensão XML:** O modelo **DeepSeek** demonstrou a maior capacidade de interpretar a estrutura XML sem ignorar as restrições técnicas.
* **Verbosidade:** Houve uma diferença significativa, com o Claude gerando quase 5x mais tokens que o ChatGPT para o mesmo objetivo.
* **Veredito:** Para prototipagem rápida e códigos complexos, o **DeepSeek** destacou-se pela precisão e fidelidade ao protocolo.
